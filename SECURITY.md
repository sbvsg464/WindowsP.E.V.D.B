# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| V5.1 release | :white_check_mark: |
| V5.0 release  | :white_check_mark: |
| V4.0 release | :x: |
| V3.1 release | :x: |
| V3.0 release | :x: |
| V2.0 release | :x: |
| V1.0 release | :x: |

## Reporting a Vulnerability
当你反馈漏洞时，应确保你处于支持安全更新的版本<br>
当你反馈漏洞时，应提供尽可能详细的描述，并且标注日期
